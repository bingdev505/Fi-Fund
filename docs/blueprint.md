# **App Name**: FinanceFlow AI

## Core Features:

- Income Tracking: Manually add and categorize income sources.
- Expense Tracking: Manually add and categorize expenses.
- Creditor Management: Track outstanding debts owed to others.
- Debtor Management: Track debts owed to you by others.
- AI Chat Input: Use natural language in a chat interface to log income, expenses, creditors, and debtors; for example, saying 'eat food for 1500 rupees' will log an expense of 1500 rupees in the food category; the LLM acts as a tool that will reason whether or not some detail is relevant.
- Financial Reporting: Generate basic reports (weekly, monthly, annual) on income vs expenses.
- AI Report Generation: Generate insights and summaries of your financial data using AI analysis; the LLM acts as a tool that will decide which data to include in its answer based on its reasoning.

## Style Guidelines:

- Primary color: Emerald Green (#28a745), symbolizing growth and wealth.
- Background color: Off-White (#f8f9fa), providing a soft and neutral backdrop.
- Accent color: Golden Yellow (#ffc107) to highlight important information and actions.
- Font: 'Roboto' (sans-serif) for body text, ensuring readability and a modern feel; 'Montserrat' (sans-serif) for headers, providing a bold and clear presentation.
- Use detailed, vibrant icons to represent categories and actions, enhancing visual appeal.
- Employ a tab-based layout for easy navigation between income, expenses, reports, and AI chat.
- Use subtle animations and transitions to provide feedback and enhance the user experience when adding data or generating reports.
